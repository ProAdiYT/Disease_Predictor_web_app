# 🧠 Diabetes Prediction Web App (Machine Learning)

A **Streamlit-based web application** that predicts whether a person is diabetic or not using Machine Learning.

This project demonstrates how to build, train, and deploy an ML model in a simple and interactive web interface.

## 🚀 Live Demo

🔗 **Try the App:**  
https://diseasepredictorwebapp-bnm3wtujbqugrf9zzugb2b.streamlit.app/


---

## 🚀 Features

- 🔍 Predict diabetes using user input
- ⚡ Fast and interactive UI with Streamlit
- 📊 Uses trained Machine Learning model
- 🧮 Input data scaling for better accuracy
- 🎯 Beginner-friendly ML deployment project

---

## 🏗️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **Machine Learning:** Scikit-learn  
- **Data Processing:** NumPy, Pandas  
- **Model Storage:** Pickle (.sav files)

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/diabetes-predictor.git
cd diabetes-predictor
```

2. Install dependencies
```
pip install -r requirements.txt
```
3. Run the app
```
streamlit run app.py
```
🧪 Input Features

The model uses the following parameters:

* Pregnancies

* Glucose Level

* Blood Pressure

* Skin Thickness

* Insulin

* BMI

* Diabetes Pedigree Function

* Age

📊 How It Works

User enters medical details

Data is scaled using pre-trained scaler

Model predicts the result

Output is displayed instantly

🎯 Output

🔴 Diabetic

🟢 Not Diabetic

🧠 Machine Learning Model

Classification model trained on diabetes dataset

Data preprocessing includes feature scaling

Model and scaler saved using Pickle
