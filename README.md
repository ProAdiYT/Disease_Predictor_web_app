# 🧠 Diabetes Prediction Web App (Machine Learning)

A **Streamlit-based web application** that predicts whether a person is diabetic or not using Machine Learning.

This project demonstrates how to build, train, and deploy an ML model in a simple and interactive web interface.

---

## 🚀 Features

- 🔍 Predict diabetes using user input
- ⚡ Fast and interactive UI with Streamlit
- 📊 Uses trained Machine Learning model
- 🧮 Input data scaling for better accuracy
- 🎯 Beginner-friendly ML deployment project

---

## 🏗️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **Machine Learning:** Scikit-learn  
- **Data Processing:** NumPy, Pandas  
- **Model Storage:** Pickle (.sav files)
